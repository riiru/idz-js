# RPG Game

RPG игра с персонажами и оружием, разработанная на JavaScript.

## Описание

Игра представляет собой пошаговую RPG битву между различными типами персонажей:
- Воины (Warriors) - сильные бойцы ближнего боя
- Лучники (Archers) - мастера дальнего боя
- Маги (Mages) - повелители магии
- Дворфы (Dwarfs) - крепкие воины с топорами
- Арбалетчики (Crossbowmen) - мастера арбалетов
- Демурги (Demourges) - маги

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```

## Команды

- `npm test` - Запуск тестов
- `npm run build` - Сборка проекта с Babel
- `npm run lint` - Проверка кода с ESLint


## Как запустить

- Запуск:
   ```powershell
   npm install
   node run-game.mjs
   ```
